Colors

  1. 140 color names, Color Names:
  2. RGB values -RED, GREEN and BLUE rgb(red, green, blue) -highest value (255)
  3. RGBA values - the opacity for a color -rgba(red, green, blue, alpha), 0.0 (fully transparent), 1.0 (not transparent at all)
  4. HEX values - #RRGGBB -black - #000000;white #ffffff 09..0a..0f..11..
  5. 3-digit hex code can only be used when both the values (RR, GG, and BB) are the same for each component
  6. HSL values hsl(hue, saturation, lightness)
  7. HSLA values -hsla(hue, saturation, lightness, alpha) 0.0 (fully transparent) and 1.0 (not transparent at all)

Backgrounds

  • background-color - color of an element
  • opacity , div {opacity: 0.3} -transparency - text + background
  • div { background: rgba(0, 128, 0, 0.3} -background only
  • background-image - covers the entire element
  • background-repeat - repeats an image both horizontally and vertically
  • background-repeat: repeat-x - horizontally; background-repeat: repeat-y; -vertically
  • only once - background-repeat: no-repeat
  • background-attachment - the background image should scroll or be fixed - : fixed/scroll
  • background-position:right top
  • background (shorthand) - all the background properties
  • body { background: #ffffff url("img_tree.png") no-repeat right top; }
  • order:color,image, repeat, attachment,position
  • background-clip -defines how far the background (color or image) should extend within an element
  • background-origin origin position of a background image

Styling Links - pseudo selectors for links

  • a:link - a normal, unvisited link
  • a:visited - a link the user has visited
  • a:hover - a link when the user mouses over it
  • a:active - a link the moment it is clicked
  • a:hover MUST come after a:link and a:visited
  • a:active MUST come after a:hover
  • text-decoration:none,underline
  • background-color
  • boxes/buttons: display: inline-block;

Combinators

  1. Descendant combinator (space) - all <p> elements inside <div> elements
  2. Child combinator (>)
  3. Next sibling combinator (+) - element that is directly after another specific element
  4. Subsequent-sibling combinator (~) - next siblings of a specified element

Pseudo Classes

used to define a special state of an element, syntax selector:pseudo-class { property: value; }

a:hover MUST come after a:link and a:visited; a:active MUST come after a:hover

Pseudo-classes can be combined with HTML classes

  • :hover pseudo-class on a <div> element
  • a <div> element to show a <p> element (like a tooltip) p { display: none; background-color: yellow; padding: 20px; } div:hover p { display: block; }
  • :first-child pseudo-class - first child of another element p:first-child { color: blue; }
  • first i element in all p p i:first-child {color: blue; }
  • :lang pseudo-class q:lang(no) {quotes: "~" "~"; }
  • input:focus {background-color: yellow; }
