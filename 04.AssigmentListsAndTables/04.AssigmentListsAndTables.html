CSS Classes and IDs

How to define a Class

  • in the HTML file - by adding the class attribute (e.g. class="personal-information") to an element
  • in the CSS file -by adding a period (.) at the end of the selector, followed by the class name (e.g. table.personal-information).

01. block elements - characteristics?

  • Display Behavior-take up the full width of their parent container by default, stacking vertically one after another.

Box Model Properties

  • Content: The main area displaying text or other child elements.
  • Padding: Space between the content and the border.
  • Border: The edge surrounding the padding.
  • Margin: Space outside the border that separates the element from other elements.
  • Width and Height - by default, width is 100% of the container, height adjusts to fit the content.
  • Nested Content: They can contain other block or inline elements.
  • Line Break:They start on a new line, pushing following content below them.

Examples: article, section, p, h1-6, div.

02. Inline elements- characteristics?

  1. No Line Break: do not start on a new line,they flow other inline elements in the same line
  2. Content-Sized - width and height -depend on their content; do not allow block-level dimensions like full width or height
  3. Box Model:Padding and margin apply only horizontally (left and right); ; vertical padding and margin don't affect layout.
  4. Contained by Block Elements: typically nested within block-level elements
  5. Purpose: <a>, <span>, <strong>, and <em>.
  6. Examples: img, strong, em, span.

  • transparent - no visible effect other than creating space
  • individually - all sides at once
  • can be defined using length units (e.g., px, em, %) or special values like auto

4. What is the padding of an Html element?

  • the space between the content and the element's border;
  • part of the CSS Box Model;
  • defines the internal spacing
  • around the content, effectively creating a buffer zone inside the element
  • inside the border, unlike the margin
  • it affects the size of the element
  • increases the space occupied within its box.
  • individually for each side (top, right, bottom, left) or all sides at once.

How Does Padding Differ from Margin?

  • in summary, padding is internal spacing, and margin is external spacing
  • padding - inside the border - around the content
  • margin -outside the border - separating elements
  • Affect on Size: padding - increases the element's box size (if box-sizing is content-box).
  • Affect on Size: margin - does not directly
  • purpose: padding: creates space between content and the border
  • purpose: margin: Creates space between elements in the layout

CSS Box Model

box that wraps around every HTML element

  • content, padding, borders and margins
  • you just set the width and height of the content area
  • to calculate the total width and height of an element, you must also include the padding and borders
  • The total width of an element should be calculated like this:
  • Total element width = width + left padding + right padding + left border + right border
  • The total height of an element should be calculated like this:
  • Total element height = height + top padding + bottom padding + top border + bottom border
  • 50px (height of content area)
  • + 20px (top padding + bottom padding)
  • + 10px (top border + bottom border)
  • = 80px (total height)
  • margin is not included in the actual size of the box.The box's total width and height stops at the border.

Floating

CSS Layout - float and clear

  • float property specifies how an element should float
  • clear property specifies what elements can float beside the cleared element and on which side
  • float property is used for positioning and formatting content
  • left-element floats to the left of its container
  • right - the element floats to the right of its container
  • none - the element inherits the float value of its parent
  • inherit - inherits the float value of its parent
  • div { float: left; padding: 15px; }
  • .div1 { background: red; }
  • .div2 { background: yellow; }
  • .div3 { background: green; }

clear property specifies what should happen with the element that is next to a floating element

none,left, right, both, inherit

  • .clearfix { overflow: auto; }
  • .clearfix::after { content: ""; clear: both; display: table; }

What is box-sizing?

  • floating boxes side by side
  • something that enlarges the width of each box (e.g. padding or borders), the box will break.
  • The box-sizing property allows us to include the padding and border in the box's total width (and height)
  • making sure that the padding stays inside of the box and that it does not break
