installation von Node.js

  1. überprüfen, ob Node.js bereits installiert ist
  2. öffne ein Terminal
  3. windows: „Eingabeaufforderung“ /„PowerShell“
  4. macOS: „Terminal“
  5. linux: je nach Distribution
  6. gib folgenden Befehl ein: node -v
  7. erscheint eine Versionsnummer - ist Node.js bereits installiert
  8. andernfalls fahre mit den nächsten Schritten fort.

installation von Node.js

  1. Offizielle Webseite
    • Besuche [https://nodejs.org](https://nodejs.org).
    • Lade das Installationspaket für dein Betriebssystem herunter (empfohlen: LTS-Version)
    • Führe das Installationsprogramm aus
  2. Paketmanager (Linux/Mac)
    • Unter Ubuntu/Debian
    • sudo apt-get update
    • sudo apt-get install nodejs
    • sudo apt-get install npm
  3. Unter macOS (mit Homebrew)
    • brew update
    • brew install node
  4. Unter Windows (mit Chocolatey)
    • choco install nodejs-lts

überprüfen der Node.js-Installation

  1. die installierte Version prüfen : node -v
  2. NPM (Node Package Manager)prüfen : npm -v

Installation von json-server

Installation über NPM

  1. npm install -g json-server - installiert den json-server global
  2. json-server -v - prüft, ob json-server erfolgreich installiert wurde
  3. Wird eine Versionsnummer angezeigt, ist json-server korrekt installiert

Einfache Verwendung von json-server

Erstelle eine Datei `db.json` in einem beliebigen Verzeichnis, zum Beispiel - > tsconfig.json

Starte den json-server, indem du in deinem Verzeichnis (wo sich `db.json` befindet) folgenden Befehl ausführst:

json-server --watch db.json --port 3000

--watch db.json` teilt dem json-server mit, dass er die Datei „db.json“ überwachen und alle Änderungen direkt übernehmen soll

--port 3000` legt den Port fest (Standard ist 3000, aber du kannst auch einen anderen wählen)

Der json-server ist nun unter `http://localhost:3000` erreichbar. Die Daten in `db.json` können z. B. unter `http://localhost:3000/todos` oder `http://localhost:3000/users` abgerufen werden.

Beispiel: index.html mit GET-Request

Erstelle in deinem Projektverzeichnis eine Datei **index.html** mit folgendem Inhalt. Dieses einfache Beispiel demonstriert, wie du mittels Fetch-API einen GET-Request an den json-server sendest und die Antwort in der Konsole ausgibst.

4.1 So führst du das Beispiel aus

  1. Stelle sicher, dass dein **json-server** über `json-server --watch db.json --port 3000` läuft.
  2. Öffne **index.html** in deinem Browser (du kannst sie auch einfach per Doppelklick öffnen).
  3. 3. Klicke auf den Button „GET /todos“.
  4. 4. Sieh dir die Entwicklertools deines Browsers (F12 → „Konsole“) an, um die abgerufenen Daten zu sehen.

Zusammenfassung

  1. Node.js installieren** (entweder über offizielle Webseite oder Paketmanager).
  2. json-server installieren** per `npm install -g json-server`.
  3. Projekt aufsetzen**:
  4. `db.json` erstellen.
  5. `json-server --watch db.json --port 3000` starten.
  6. index.html schreiben, die mit der Fetch-API eine GET-Anfrage an den json-server schickt.

Mit diesen Schritten bist du in der Lage, einfache REST-ähnliche Endpoints mit dem json-server zu erstellen und anschließend per JavaScript (Fetch API) Daten abzurufen.
